Инструкция по установке

шаг 1: Откройте Chrome Web Store
Перейдите на страницу расширения A-PASS в магазине Chrome.
шаг 2: Нажмите «Установить»
На странице расширения нажмите синюю кнопку «Установить».
Шаг 3: Подтвердите установку
Всплывет окно с запросом на разрешения для расширения. Для завершения нажмите кнопку «Установить расширение».
Шаг 4: Готово!
Расширение будет установлено, и его значок появится справа в адресной строке браузера.
Поддерживаемые браузеры:
Opera
Google
Chrome
Microsoft
Edge
Yandex
Требования:
  • Версия браузера не ниже 88
  • Windows 10+, macOS 10.14+ или Linux Ubuntu 16.04+
  • Разрешение на доступ к данным сайтов
  • Разрешение на уведомления
Безопасность:
A-Pass Key генерирует криптографическое доказательство, которое может быть передано по открытым каналам без разглашения чувствительных данных.
